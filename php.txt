Все типы данных в PHP:
1. integer - целое число
2. float - число с плавающей точкой (22.5)
3. boolean - логический тип данных (true или false)
4. string - строка
5. array - массив
6. object - объект (конструкция, объединяющая несколько разнотипных переменных и методы их обработки)
7. resource - ресурс (дескриптор, позволяющий апперировать тем или иным ресурсом)
8. null - переменная не была инициализирована
9. collable - некоторые функции PHP, которые могут принимать в качестве аргументов другие функции
10. void - отсутствие типа
11. iterable - используется для обозначения перечисляемого параметра или возвращаемого значения функции
12. mixed - значение может иметь любой тип данных
13. never - обозначение возвращаемого типа
